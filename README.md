# Node JWT Authentication

## Pre Requirements

node ```v5.12.0```

npm ```3.8.6```

mongodb ```3.0.14```


## Install Dependencies

clone the project from https://github.com/shubhambatra/node-jwt-auth.git 

and type following command
```sh 
$ cd node-jwt-auth

npm install
npm run server
``` 
open http://127.0.0.1:8000/
